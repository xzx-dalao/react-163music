# React技术栈 网易云音乐web版项目实战

## 项目简介
### 技术栈
 `react` + `redux` + `react-router` + `styled-components` + `axios` + `ant design` + `react-redux` + `redux-thunk` `immutable` + `redux-immutable` + `craco` 等来开发web版的网易云音乐官网。
